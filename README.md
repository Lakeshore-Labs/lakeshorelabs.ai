# Lakeshore Labs

Modern Hugo-based website for lakeshorelabs.ai

## Quick Start

```bash
# Install Hugo Extended v0.147.0+
# Clone repository
git clone [repository-url]
cd hugo

# Development
hugo server

# Production build
hugo --minify
```
